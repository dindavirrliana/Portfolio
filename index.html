Project 1 : Sentiment Analysis

Analysis Sentiment of Covid-19 Vaccine on Social Media Twitter
The project was done during the "Data Analysis using Machine Learning" course program held by Institut Teknologi Telkom Purwokerto in KMMI Program

Background: The data used is 1000 tweets that have been scrapped which contain tweets with the keyword vaccinecovid19 within 1 week, This analysis is used to find public opinion sentiment towards the COVID-19 vaccine

Result:

1. The public had positive sentiments/responses of 457 tweets, 333 neutral tweets, and 210 negative tweets about vaccine COVID-19 based on the results of sentiment classification on Twitter social media

2. The Naive Bayes classification algorithm produces an accuracy value of 84.1% for classifying public sentiment
